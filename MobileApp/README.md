# Mobile APP

---
#### Programming language
- Typescript
    - React native (framework)
    
---
#### How to use:
Go to directory
```
cd App
```

Install dependency
```
npm install package.json
```


Start application in developer mode
```
npm start
```

Install expo on mobile phone and scann the qr code for real live preview
